Installation instructions

NOTE: Leave the expression asm=1 out if it appears that the program is not otherwise compiling properly.

Unix-compatible OS:es (Linux, BSD, OpenSolaris etc.)

Extract the source package with tar and gunzip and then run 
  $ make clean; make asm=1

  $ cp needle needle.bk <hakemisto, jossa ajat XBoardia Needlen kanssa>
In order to deploy in Unix-compatible systems with XBoard: 
  $ xboard -fcp needle -size Small -tc 5 -inc 10 &
Size, time control and increment are just examples, Needle accepts all time control types.

Micro$oft Windows

In order to compile the source package in Micro$oft Windows-systems by MSYS/MinGW or Cygwin: 
  $ make clean; make mswindows=1 asm=1

  $ cp needle<version>.exe needle.bk <Your WinBoard directory>
In order to deploy in Micro$oft Windows systems with WinBoard:

Edit 'winboard.ini' and add 'needle<version>.exe' into sections /firstChessProgramNames and /secondChessProgramNames. This also applies to the binary package.
